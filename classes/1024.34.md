This defines the response from a Level II node for a [CLASS1.PROTOCOL, Type=33, VSCP2_TYPE_PROTOCOL_GET_MATRIX_INFO](./class1.protocol.md#type34) event.

 | Byte   | Description    |
 | :----: | -----------    |
 | 0-15   | GUID for node  |
 | 16     | DM row size    |
 | 17     | DM number of rows |
 | 18-21  | Register start for DM |
