This is for controlling playback functionality 

 | Data byte | Description                                                        | 
 | :---------: | -----------                                                        | 
 | 0         | Function (See below)                                               | 
 | 1         | Zone for which event applies to (0-254). 255 is all zones.         | 
 | 2         | Sub-zone for which event applies to (0-255). 255 is all sub-zones. | 

##### Function Codes

 | Code | Description         | 
 | :----: | -----------         | 
 | 0    | Stop                | 
 | 1    | Pause               | 
 | 2    | Play                | 
 | 3    | Forward             | 
 | 4    | Rewind              | 
 | 5    | Fast Forward        | 
 | 6    | Fast Rewind         | 
 | 7    | Next Track          | 
 | 30   | Previous Track      | 
 | 31   | Toggle repeat mode  | 
 | 32   | Repeat mode ON      | 
 | 33   | Repeat mode OFF     | 
 | 34   | Toggle Shuffle mode | 
 | 35   | Shuffle ON          | 
 | 36   | Shuffle mode OFF    | 
 | 37   | Fade in, Play       | 
 | 38   | Fade out, Stop      | 

Appropriate **CLASS1.INFORMATION** events should be sent from the controlled device as response to this event.
