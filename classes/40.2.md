This is typically for navigation functions or DVD controls 

 | Data byte | Description                                                        | 
 | :---------: | -----------                                                        | 
 | 0         | Function                                                           | 
 | 1         | Zone for which event applies to (0-254). 255 is all zones.         | 
 | 2         | Sub-zone for which event applies to (0-255). 255 is all sub-zones. |

 | :----:    | -----------                                                                            |

 | 10      | 10+ key                                                                                | 
 | 20      | OK                                                                                     | 
 | 21      | Left                                                                                   | 
 | 22      | Right                                                                                  | 
 | 23      | Up                                                                                     | 
 | 24      | Down                                                                                   | 
 | 25      | Menu                                                                                   |
